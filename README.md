## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/BrijeshShinde/Python_Assignment.git
   ```

2. Navigate to the folder:

   ```bash
   cd Python_Assignment
   ```

3. Open Jupyter Notebook:

   ```bash
   jupyter notebook
   ```

4. Run the `.ipynb` files
