# 🚀 Python Projects Collection

This repository contains a collection of beginner-to-intermediate level Python projects. Each project focuses on solving real-world problems and strengthening core Python concepts like file handling, OOP, APIs, and data processing.

---

## 📂 Project Structure

```
Projects/
│
├── contact/               # Contact management system
├── Employee_system/      # Employee record handling system
├── Hospital_management/  # Hospital operations management
├── mark_analyzer/        # Student marks analysis tool
├── movie_tracker/        # Track and manage movies
├── weather_logger/       # Weather data logging using API
├── README.md             # Project documentation
```

---

## 🔥 Projects Overview

### 📇 Contact Management System

* Add, update, delete contacts
* Search contacts easily
* File-based storage

### 👨‍💼 Employee System

* Manage employee records
* CRUD operations (Create, Read, Update, Delete)
* Useful for HR basics

### 🏥 Hospital Management System

* Patient record management
* Doctor and appointment tracking
* Basic healthcare workflow simulation

### 📊 Mark Analyzer

* Analyze student marks
* Calculate average, grades, and performance
* Useful for academic insights

### 🎬 Movie Tracker

* Add and track watched movies
* Store ratings and reviews
* Personal entertainment manager

### 🌦️ Weather Logger

* Fetch weather data using API
* Store logs for analysis
* Helps understand API integration

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/BrijeshShinde/Python_Assignment/tree/master/Projects
   ```

2. Navigate to project folder:

   ```bash
   cd Projects
   ```

3. Run any project:

   ```bash
   python filename.py
   ```
